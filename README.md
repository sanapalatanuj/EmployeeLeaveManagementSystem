# Employee Leave Management System

## Overview

The Employee Leave Management System is a Spring Boot REST API application developed to manage employees, departments, leave types, leave requests, and leave balances efficiently. The system provides functionalities for leave application, approval, rejection, and leave balance tracking.

This project was developed as part of my internship to gain practical experience in Java backend development, Spring Boot, RESTful APIs, database management, and software development best practices.


## Features

- Employee Management
  - Add Employee
  - View Employees
  - Update Employee
  - Delete Employee

- Department Management
  - Add Department
  - View Departments
  - Update Department
  - Delete Department

- Leave Type Management
  - Add Leave Type
  - View Leave Types
  - Delete Leave Type

- Leave Request Management
  - Apply for Leave
  - View Leave Requests
  - Update Leave Requests
  - Delete Leave Requests
  - Approve Leave Requests
  - Reject Leave Requests

- Leave Balance Management
  - Manage Employee Leave Balances

- Validation and Exception Handling

- Swagger API Documentation

- MySQL Database Integration

1. Clone the repository.
2. Configure MySQL database in application.properties.
3. Run the application.
4. Access Swagger UI:
http://localhost:8080/swagger-ui/index.html
